# 剪贴板历史管理 产品规格说明

- **优先级**：P0
- **对应设计**：`Docs/design/features/02-Clipboard-History.md`
- **版本**：v1.0

---

## 一、功能概述

为 ToolBox 增加全系统剪贴板历史监听与管理能力。后台持续监听 Windows 剪贴板变化，自动记录文本、图片、文件路径三种类型的内容。用户可通过热键 `Ctrl+Shift+V` 唤起剪贴板浮窗浏览历史，支持搜索、置顶、收藏，点击条目即可粘贴到当前焦点应用。同时在工具台增加独立的剪贴板管理页面。该功能补齐了 ToolBox 作为效率工具的基础能力，并与截图系统和 AI 助手形成协同。

## 三、验收标准

### 剪贴板监听与记录

- [x] **AC1.1**：ToolBox 启动后自动开始监听系统剪贴板，任何文本/图片/文件复制操作均被记录到历史
- [x] **AC1.2**：连续复制相同内容不产生重复记录（基于内容哈希 SHA256 去重）
- [x] **AC1.3**：历史记录上限为 500 条（可配置），超出后最旧条目自动清理（含关联缩略图文件）
- [x] **AC1.4**：截图后自动复制到剪贴板的图片被正确记录，不因截图窗口句柄而漏记

### 浮动面板 (ClipboardPopup)

- [x] **AC2.1**：按下 `Ctrl+Shift+V` 时在屏幕中央弹出 ClipboardPopup（360×480px, Topmost），显示最近历史
- [x] **AC2.2**：面板顶部搜索框支持实时过滤（输入关键词即时匹配文本内容），输入响应时间 ≤ 100ms
- [x] **AC2.3**：提供"文本/图片/文件"三个筛选 Tab（FilterTabStyle），切换时列表即时刷新
- [x] **AC2.4**：文本条目显示前 80 字符 + 时间戳；图片条目显示 48×48 缩略图 + 时间戳；文件条目显示文件名 + 路径摘要
- [x] **AC2.5**：点击条目 → 内容写入系统剪贴板 → 自动向当前前台窗口发送 Ctrl+V 粘贴
- [x] **AC2.6**：每个条目支持 ⭐ 收藏 / 📌 置顶 / 🗑 删除操作；置顶条目排在最前
- [x] **AC2.7**：面板支持键盘导航：↑↓ 切换条目，Enter 粘贴，ESC 关闭
- [x] **AC2.8**：底部状态栏显示"共 X 条 · 已置顶 Y · 已收藏 Z"

### 工作台剪贴板页面

- [x] **AC3.1**：工作台左侧侧边栏新增"📋 剪贴板"第 6 个导航项（位于截图页之后）
- [x] **AC3.2**：ClipboardView 布局与 HistoryView 一致——顶部筛选栏 + 网格/列表视图切换 + 分页（每页 20 条）
- [x] **AC3.3**：支持时间筛选（today/week/month/all）、类型筛选（文本/图片/文件）
- [x] **AC3.4**：点击条目右侧详情面板显示完整内容 + 来源应用 + 操作按钮

### 隐私与协作

- [x] **AC4.1**：设置页"📋 剪贴板"Section 中可配置"忽略的应用列表"，来自这些应用的复制不记录
- [x] **AC4.2**：提供"暂停监听 5 分钟"快捷操作（托盘菜单或热键）
- [x] **AC4.3**：ClipboardPopup 中选中条目可"发送到 AI 助手"→ 跳转工作台 assistant 页并填入内容

## 四、功能范围

### In Scope（本期实现）

- Windows 剪贴板文本/图片/文件路径监听（Win32 Clipboard Chain）
- 内容 SHA256 去重
- 浮动剪贴板面板（Ctrl+Shift+V 唤起）
- 搜索、类型筛选、置顶、收藏、删除
- 工作台独立剪贴板页面（分页 + 时间筛选）
- 点击粘贴到前台应用
- 忽略应用列表 + 暂停监听
- 发送到 AI 助手
- 剪贴板历史上限可配置
- LLM 工具 `search_clipboard` / `get_clipboard`

### Out of Scope（本期不实现）

- 剪贴板历史云同步
- 富文本格式保留（仅记录纯文本）
- 图片原图存储（仅存 200×200 缩略图）
- 剪贴板内容全文搜索 10000+ 条（当前 JSON 方案适用 ≤500 条）
- HTML/RTF 格式内容记录
- 多设备剪贴板共享

## 五、交互流程

### 主流程：复制 → 回溯 → 粘贴

```
1. 用户在浏览器中复制一段文字 → ClipboardMonitor 监听到变化
2. 内容去重（与上一条 SHA256 对比）→ 存入 ClipboardStore → JSON 持久化
3. 30 分钟后，用户按 Ctrl+Shift+V → ClipboardPopup 弹出
4. 用户在搜索框中输入关键字 → 列表实时过滤
5. 用户找到目标条目 → 点击 → 内容写入剪贴板 + Ctrl+V 粘贴到前台应用
6. ClipboardPopup 自动关闭
```

### 隐私暂停流程

```
1. 用户在 1Password 中复制密码 → （该应用已在忽略列表中）→ 不记录
2. 用户右键托盘 → 选择"暂停剪贴板监听 5 分钟"→ 5 分钟内所有复制不记录
3. 5 分钟后自动恢复 → 托盘提示"剪贴板监听已恢复"
```

## 六、非功能需求

### 性能

- 剪贴板监听对系统输入延迟增加 ≤ 1ms（WM_DRAWCLIPBOARD 中仅排队，异步处理）
- ClipboardPopup 从按键到显示 ≤ 200ms
- 500 条历史搜索过滤 ≤ 50ms
- 缩略图生成（200×200 JPEG）≤ 100ms/张

### 安全

- 剪贴板历史数据存储在本地用户目录，不联网
- 支持应用级忽略列表，防止密码管理器等应用的内容被记录
- 删除的历史条目不可恢复（直接清理 JSON 记录 + 缩略图文件）
- "清空全部历史"操作需要二次确认

### 可访问性

- ClipboardPopup 完全支持键盘操作（↑↓Enter ESC）
- 搜索框自动获取焦点
- 文本截断显示时有 Tooltip 显示完整内容

### 国际化

- UI 文案通过语言包资源字典切换
- 时间戳显示格式跟随系统区域设置
- 搜索不区分中英文大小写

## 八、竞品参考

| 竞品 | 实现方式 | ToolBox 借鉴点 |
|------|---------|---------------|
| **Raycast Clipboard** | 可视/可搜/无限历史，支持图片预览和收藏 | 浮动面板搜索 + 类型 Tab 筛选 + 收藏置顶体系 |
| **uTools 剪贴板插件** | 搜索框唤起，类型分组，自动粘贴 | 热键唤起浮窗 + 点击自动粘贴的交互闭环 |
| **Windows Win+V** | 系统内置，Win11 支持固定/emoji/GIF | 使用不同热键（Ctrl+Shift+V）避免冲突，共存互补 |
| **ShareX** | 截图后自动复制 → 被 ShareX 历史记录 | 与截图系统协同——截图后自动入剪贴板历史 |

ToolBox 差异化：(1) 与截图/AI 深度协同——剪贴板内容可直接发送到 AI 助手分析；(2) 工作台内独立管理页面——不只是浮窗，还支持分页/时间筛选/详情面板；(3) 隐私优先——忽略应用列表 + 暂停监听的细粒度控制。
